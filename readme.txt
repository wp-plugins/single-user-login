== Description ==

force user login into wordpress only from one browser at the same time
the last login action will kick out the already logged  user
this is a security plugin for wordpress,install this plugin to  avoid simultaneously login

== Installation ==

 

1. Upload `single-user-login` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress 
3. done
